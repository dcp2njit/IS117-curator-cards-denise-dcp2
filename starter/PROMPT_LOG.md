## Prompt 3

- **Phase: Exploration**
- **Prompt name: Concept Map**
- **What I sent:**

```
Create a concept map showing how these three design movements relate to each other: 
1. Bauhaus (1919–1933) 
2. International Typographic Style / Swiss Style (1950s–1970s) 
3. Grunge Typography
Show: key figures, core principles, what each movement inherited from or rejected about the previous one. Use plain text with indentation — no images.
```

- **What I got back (summary): A plain text concept map showing the lineage, inheritance, and rejection of the three movements in terms of the evolution of history of modern design.**

- **What I did with it: i pinned the core principles associated with each style and the key figures**

```

For each term provide: - The term - A one-sentence definition (max 15 words) - Which movement(s) it applies to Only include terms a design professional would use.
